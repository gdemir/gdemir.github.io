Konsol üzerinden [Sublime Text](http://www.sublimetext.com/) editörünü kurmak için aşağıdaki komutlar yeterlidir.

	sudo add-apt-repository ppa:webupd8team/sublime-text-2
	sudo apt-get update
	sudo apt-get install sublime-text
